# Zenith

**Zenith** is an open-source astrophotography image processing application designed for efficient processing and analysis of astronomical images.

The goal of Zenith is to provide a modern, flexible, and high-performance tool for astrophotographers.

## Features

Currently under development.

Planned features include:

- FITS, TIFF, XISF image support
- Calibration, registration and integration tools
- Denoising and deconvolving tools
